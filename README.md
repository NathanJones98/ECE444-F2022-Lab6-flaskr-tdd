# What are the pros and cons of TDD?

Pros: 

-Only need to write code that is needed.

-More modular design

-Easier to maintain

-Less Debugging



Cons:

-Tests help show errors, but still require debugging

-TDD is a slow process

-Tests need to be maintained when requirements change
